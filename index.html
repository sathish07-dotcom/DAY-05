CSS (Cascading Style Sheets) is used to style HTML elements.

There are three main types of CSS:

1.Inline CSS

Written directly within the HTML element using the style attribute.Affects only that specific element.

2. Internal CSS

Written within a style tag in the head section of an HTML document.Affects the elements of the page it is written on.

3. External CSS

Written in a separate .css file and linked to the HTML file using the tag in the section. Allows reusability across multiple web pages.
